# ForumCreator - Application de création de forums

**Accéder à l'application :** https://forum-creator.onrender.com

## Description du Projet
Création de forums par les utilisateurs qui permettent d'envoyer des messages lorsque l'on est connecté.
On peut:
* Créer des forums
* Supprimer nos forums

* Envoyer des messages dans ces forums
* Supprimer nos messages

* Modifier son nom d'utilisateur
* Modifier sa photo de profil
* Supprimer son compte

* Changer le thème de l'application (clair/sombre)
* Change la langue de L'application (francais/anglais)

On ne peut pas:
* Supprimer les forums/messages des autres à moins d'être un admin
* Supprimer le compte des autres
* Accéder au profil d'autres utilisateurs

PS : Après la création d'un compte, il faut se connecter. Ceci n'est pas un bug.
